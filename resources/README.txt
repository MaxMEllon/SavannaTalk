SavannaTalk

AfreecaTVで棒読みちゃんがコメントを読み上げるようにします。

# 使い方

* Chrome 64ビット版をダウンロードしてください。（通常、32ビット版が入っていますので、64ビット版をインストールし直してください）
	* https://www.google.com/chrome/browser/desktop/index.html
	* 「別のプラットフォーム向けの Chrome をダウンロード」を選び「Windows 10/8.1/8/7 64-bit」をインストールしてください。
* 棒読みちゃんを起動してください。
* 棒読みちゃんの設定でSocket通信の「Socket通信を使う」がTrueになっていることを確認してください。
* SavannaTalk.exeを起動してください。
* 放送URLを入力してOKを押してください。
* 以降、コメントが読み上げられます。
